==================================================
TROUBLESHOOTING
==================================================
1. Database Connection Error:
   - Check MySQL is running
   - Verify credentials in /config/database.php
   - Make sure database 'digital_tarai' exists

2. Cannot Access Setup Page:
   - Ensure PHP is running
   - Check XAMPP is started
   - Verify file exists in correct location

3. Login Not Working:
   - Clear browser cookies/cache
   - Check database has users table
   - Verify credentials are correct

4. File Upload Error:
   - Check /public/uploads/ folder exists
   - Verify folder has write permissions
   - Check max file size limit
